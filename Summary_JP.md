# Unity MCP インテグレーション

これは Model Context Protocol (MCP) に基づく Unity 実装であり、AI アシスタント（Cursor、Claude Code、OpenCode）が Unity エディターとランタイムアプリケーションを直接制御できるようにします。

## 特徴

### 純粋な C# 実装
Unity で完全に開発され、Unity C# 構文と互換性があります。

### MCP プロトコル
MCP プロトコルドキュメントで定義されたほぼすべての機能をサポートしています（Prompts を除く）。

### ツールの開発と拡張
- 80 種類の組み込みエディターツールを提供
- プロジェクト要件に応じてツールを拡張するためのシンプルなツール定義方法を提供
- プロトコルは複数のデータ型をサポート：
  - 基本データ型
  - Unity 組み込みのジオメトリタイプ
  - 配列型
  - ユーザー定義のカスタム型（制約あり）

### AI 支援コーディングサポート
AI による二次開発と拡張のための AGENTS.md ファイルを提供。

### モード
エディターモードとランタイムモードの両方をサポート。
