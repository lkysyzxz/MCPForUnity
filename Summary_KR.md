# Unity MCP 통합

이것은 Model Context Protocol (MCP)를 기반으로 한 Unity 구현으로, AI 어시스턴트(Cursor, Claude Code, OpenCode)가 Unity 에디터와 런타임 애플리케이션을 직접 제어할 수 있게 합니다.

## 특징

### 순수 C# 구현
Unity에서 완전히 개발되었으며 Unity C# 구문과 호환됩니다.

### MCP 프로토콜
MCP 프로토콜 문서에 정의된 거의 모든 기능을 지원합니다 (Prompts 제외).

### 도구 개발 및 확장
- 80개의 기본 제공 에디터 도구 제공
- 프로젝트 요구 사항에 맞게 도구를 확장할 수 있는 간단한 도구 정의 방법 제공
- 프로토콜은 다양한 데이터 유형을 지원:
  - 기본 데이터 유형
  - Unity 내장 지오메트리 유형
  - 배열 유형
  - 사용자 정의 커스텀 유형 (제약 조건 있음)

### AI 지원 코딩 지원
AI를 통한 2차 개발 및 확장을 위한 AGENTS.md 파일 제공.

### 모드
에디터 모드와 런타임 모드를 모두 지원합니다.
